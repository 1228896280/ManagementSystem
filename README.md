### 功能
- 查询员工

	- 表联合查询

- 增加员工

	- 数据校验

	- 前后端数据库同时校验

	- 前端数据合法验证

- 更新员工

	- 数据合法验证

- 删除员工

    - 单个批除

    - 批量删除

### 核心技术栈

项目框架：Spring/Spring MVC/Mybatis

基础工具：Maven/Tomcat/MySQL/JDK1.8

前端技术：jQuery/Bootstrap/Ajax

其他技术：Java高级技术/Hibernate-validator/ JSR303/ Jackson
