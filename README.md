### 功能
- 查询员工

	- 表联合查询

- 增加员工

	- 数据校验

	- 前后端数据库同时校验

	- 前端数据合法验证

- 更新员工

	- 数据合法验证

- 删除员工

    - 单个批除

    - 批量删除

### 技术栈

- Java高级技术:hibernate-validator,JSR303

- 框架:Spring,Spring Mvc ,Mybatis，bootstrap

- 前端:html.css.js,jquery

- 数据交换:ajax,mysql
